# wedding-invitation

Invitation car site web pour un mariage.

## Présentation

Projet d’invitation de mariage en ligne, conçu comme une expérience élégante, fluide et responsive. L’application présente les détails de la cérémonie, le lieu, la carte interactive et les informations de contact dans un style moderne et raffiné.

## Fonctionnalités

- Interface élégante et responsive
- Section de présentation du couple
- Détails de l’événement (date, heure, lieu, adresse)
- Carte interactive pour le lieu de réception
- Animations fluides avec Framer Motion
- Compatible avec un déploiement rapide sur Vite / Netlify

## Stack technique

- React 19
- Vite
- Framer Motion
- CSS Modules
- React Icons

## Installation

1. Cloner le projet
   ```bash
   git clone https://github.com/mahdi-toumi-1/wedding-invitation.git
   cd wedding-invitation
   ```

2. Installer les dépendances
   ```bash
   npm install
   ```

3. Lancer le projet en local
   ```bash
   npm run dev
   ```

## Scripts disponibles

- `npm run dev` : démarre le serveur de développement
- `npm run build` : génère la version de production
- `npm run preview` : prévisualise la version buildée
- `npm run lint` : exécute la vérification du code
